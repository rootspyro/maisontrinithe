# Maisontrinithe
This is the landing page of the Maisontrinithe company.

## Installation

```bash

git clone https://github.com/rootspyro/Maisontrinithe
cd Maisontrinithe
npm install
npm run dev 

# Output: 
# astro dev --port 3000 --host 0.0.0.0
# Local     http://localhost:3000/
# Network   http://x.x.x.x:3000/

```
